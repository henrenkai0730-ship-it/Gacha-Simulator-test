# 抽卡模拟器 (Gacha Simulator)

一个简单的Python抽卡模拟器，支持保底机制、UP概率、十连抽等功能。

## 功能特性

- **基础概率**：六星2%、五星8%、四星50%、三星40%
- **保底机制**：超过50次未出六星后，每次数+2%（其他按权重缩减）
- **UP机制**：六星中50%为当期UP；连续2次非UP后，下次必UP
- **抽卡模式**：单抽、十连抽
- **记录保存**：自动保存抽卡历史到文件
- **可视化输出**：用星星显示星级，每轮结果明细

## 安装与运行

### 环境要求
- Python 3.x

### 运行步骤
1. 下载 `gacha_simulator.py`
2. 打开终端，进入文件目录
3. 运行命令：`python gacha_simulator.py`

## 使用方法

启动后，选择模式：
- `1`：单抽模式（输入抽卡次数）
- `2`：十连抽模式（自动10次）
- `3`：查看抽卡记录
- `q`：退出

### 示例输出

```
本轮抽卡 10 次结果：
 1. ⭐⭐⭐⭐ 四星
 2. ⭐⭐⭐ 三星
 3. ⭐⭐⭐⭐⭐⭐ 六星（UP）
...
```

## 规则说明

### 概率分布
- 六星：2%
- 五星：8%
- 四星：50%
- 三星：40%

### 保底规则
- 六星保底：50次未出后，每次数+2%，最高100%
- UP保底：连续2次六星非UP，下次必为UP

### UP机制
- 六星出货时，50%概率为当期UP
- 保底触发时，必定为UP

## 文件说明

- `gacha_simulator.py`：主程序
- `gacha_history.txt`：抽卡历史记录（自动生成）
